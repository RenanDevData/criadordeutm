Como usar o Gerador de UTM

Este guia explica o que é cada parâmetro UTM e como preencher corretamente na ferramenta.

1. O que são UTMs?

UTMs são códigos adicionados no final de uma URL para identificar:

  • De onde veio o tráfego
  • Qual campanha trouxe o clique
  • Qual anúncio ou variação performou melhor

2. Explicação de cada parâmetro

utm_source — De onde vem o usuário

É a plataforma que originou o tráfego.

Exemplos: google, instagram, tiktok, facebook

utm_medium — Tipo de tráfego

Identifica a "categoria" do tráfego.

Exemplos: cpc, paid_social, email, organic

utm_campaign — Nome da campanha

O nome da ação ou campanha de marketing.

Exemplos: black_friday_2025, lançamento_curso_excel

utm_term — Palavra-chave (opcional)

Usado principalmente em Google Ads.

Exemplo: curso_excel_online

utm_content — Variação do anúncio (opcional)

Perfeito para identificar cada peça criativa.

Exemplos: criativo_1, video_long, imagem_azul

utm_id — ID da campanha

Recomendado para rastreamento avançado.

Exemplo: campanha_22

pmkt — Parâmetro extra (se aplicável)

Campo para uso livre caso queira adicionar algum identificador próprio.

3. Como usar a ferramenta

  1. Digite sua URL base (ex.: https://meusite.com/produto).
  2. Preencha ao menos utm_source e utm_medium.
  3. Preencha os demais campos conforme sua campanha.
  4. Clique em Gerar UTM.
  5. O sistema normaliza tudo (minúsculo, sem acento, com _).
  6. Copie o link final e use na campanha.

4. Exemplo de URL gerada

https://meusite.com/produto? utm_source=google& utm_medium=cpc& utm_campaign=black_friday& utm_term=curso_excel& utm_content=criativo_1

5. Boas práticas rápidas

  • Use sempre minúsculas.
  • Evite espaços — use _.
  • Seja claro e consistente.
  • Evite nomes longos demais.
  • Não use acentos.
